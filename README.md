# 云借阅图书管理系统

一个纯前端的图书借阅管理系统，支持图书管理、用户管理、借阅归还、书籍推荐等功能。

## 功能特性

- **图书管理** — 图书入库、编辑、删除，支持 ISBN、分类、库存管理
- **用户管理** — 读者注册、信息维护、借阅记录追踪
- **借阅管理** — 借书、还书、逾期提醒、借阅历史
- **书籍推荐** — 热门排行、新书推荐、管理员置顶推荐
- **数据统计** — 可视化图表展示借阅趋势、热门图书

## 技术栈

- 纯前端 SPA（HTML + CSS + JavaScript）
- Chart.js 图表库
- localStorage 数据持久化

## 快速开始

直接在浏览器中打开 `index.html` 即可使用。
